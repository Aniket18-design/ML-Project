# ML-Project

This project implements a production-ready ML pipeline for fare prediction using PySpark.

## 🚀 Features
- Temporal Train/Test split
- 5 regression models
- RMSE evaluation
- Best model selection
- Dockerized pipeline
- Tableau dashboards

## 🧠 Models Used
- Linear Regression
- Decision Tree
- Random Forest
- Gradient Boosted Trees
- Generalized Linear Regression

## 📦 How to Run

### Local

bash scripts/setup_environment.sh
python scripts/run_pipeline.py
